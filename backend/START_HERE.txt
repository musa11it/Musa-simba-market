🔌 API ENDPOINTS (Sample)
═══════════════════════════════════════════════════════════════════════════

Authentication:
   POST   /api/auth/login                    → Login user
   POST   /api/auth/register                 → Create account
   GET    /api/auth/me                       → Current user

Super Admin:
   GET    /api/super-admin/dashboard         → System stats
   GET    /api/super-admin/branches          → List branches
   POST   /api/super-admin/branches          → Create branch
   POST   /api/super-admin/admins            → Create admin
   GET    /api/super-admin/analytics         → System analytics

Branch Admin:
   GET    /api/branch-admin/dashboard        → Branch stats
   GET    /api/branch-admin/products         → List products
   POST   /api/branch-admin/products         → Add product
   GET    /api/branch-admin/orders           → List orders
   GET    /api/branch-admin/analytics        → Branch analytics

Public:
   GET    /api/products                      → List all products
   POST   /api/orders                        → Create order
   GET    /api/orders                        → List orders
   GET    /api/analytics                     → Public analytics

Full API List: See API_DOCS.md (25+ endpoints)

💡 COMMON TASKS
═══════════════════════════════════════════════════════════════════════════

1. START THE SERVER:
   npm run dev              (Development with auto-reload)
   npm start                (Production)

2. LOGIN:
   Visit: http://localhost:5000/login.html
   Use default credentials

3. CREATE A BRANCH:
   Super Admin → Branches → Add Branch → Fill Details

4. CREATE BRANCH ADMIN:
   Super Admin → Admins → Create Admin → Select Branch

5. ADD PRODUCTS:
   Branch Admin → Products → Add Product → Fill Details

6. MANAGE ORDERS:
   Branch Admin → Orders → View & Update Status

7. VIEW ANALYTICS:
   Both → Analytics → Select Period

🆘 TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════════════

Cannot start server?
   → Check Node.js is installed: node --version
   → Check MongoDB is running
   → Check port 5000 is available

Cannot login?
   → Run: npm run init (resets database)
   → Check default credentials

Dashboard blank?
   → Open browser console (F12)
   → Check for JavaScript errors
   → Check API_BASE_URL

More help?
   → See SETUP_GUIDE.md (Troubleshooting section)
