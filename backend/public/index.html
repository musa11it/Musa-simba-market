📋 First-Time Setup Steps

1

Ensure MongoDB is Running

MongoDB should be running on mongodb://localhost:27017

2

Initialize Database

Run in terminal: npm run init

3

Start the Server

Run in terminal: npm run dev or node src/server.js

4

Login

Use the default credentials above. Click either dashboard to proceed.

5

Create Your First Branch

As Super Admin: Go to "Branches" → "Add Branch" → Fill in details
